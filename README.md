Consultando la Base de Datos de una Universidad

* Recursos del Examen (GitHub)
Los materiales para el examen están alojados en el repositorio del curso. Por favor, descarguen ambos archivos antes de comenzar:

1.	Base de Datos (universidad.db):
o	Descargar archivo .db aquí
o	Instrucción: Deben pulsar el botón de descarga (Download raw file) que aparece a la derecha.
1.	Cuaderno del Examen (Examen.ipynb):
o	Descargar Notebook aquí
o	Instrucción: Abran el enlace y usen el botón de descarga en la parte superior derecha (icono de flecha hacia abajo). No guarden la página web como HTML.
 
1.	Entorno: Coloquen ambos archivos (Examen.ipynb y universidad.db) en la misma carpeta de su equipo.
2.	Ejecución: Abran el cuaderno en VS Code, Jupyter Lab o Google Colab (si usan Colab, recuerden subir la .db al entorno temporal).
3.	Estructura del Reto (15 Puntos):
o	Sección 1 (Gestión / DML): Creación de cursos y matrículas manuales (5 pts).
o	Sección 2 (Consultas / SQL): Reportes usando JOINS, GROUP BY y Subconsultas (6 pts).
o	Sección 3 (Analítica / Pandas): Análisis de riesgo y visualización de datos (4 pts).
1.	Entrega: Al finalizar, guarden los cambios en su cuaderno y suban el archivo .ipynb en este espacio de entrega
